# AI Quiz Platform

Live Demo: https://ai-quiz-platform-gilt.vercel.app

## Overview

AI Quiz Platform is a modern web application that generates and delivers interactive quizzes based on user-selected topics. The application focuses on providing a clean user experience, dynamic content generation, and real-time interaction. It is built using a modern frontend architecture and deployed on Vercel for global access.

## Features

* Topic-based quiz generation
* Interactive question interface
* Real-time score calculation
* Responsive and modern UI
* Fast loading and optimized performance
* Public deployment for live usage

## Tech Stack

This project is built with:

* Vite
* TypeScript
* React
* shadcn-ui
* Tailwind CSS

## Deployment

The application is deployed on Vercel.

Live URL:
https://ai-quiz-platform-gilt.vercel.app

## Getting Started (Run Locally)

1. Clone the repository

```
git clone https://github.com/gt-vibu/ai-quiz-platform-.git
```

2. Navigate to the project folder

```
cd ai-quiz-platform-
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

5. Open in browser

```
http://localhost:5173
```
